# Imágenes Requeridas

Este directorio contiene placeholders. Reemplaza con tus imágenes reales.

## Imágenes Necesarias:

### Logo
- `comedica-logo-white.png` - Logo blanco para header (recomendado: 200x60px, PNG con transparencia)

### Login Carousel
- `login/slide-1.jpg` - Imagen 1 del carousel de login (recomendado: 1920x1080px)
- `login/slide-2.jpg` - Imagen 2 del carousel de login (recomendado: 1920x1080px)
- `login/slide-3.jpg` - Imagen 3 del carousel de login (recomendado: 1920x1080px)

## Para reemplazar:
1. Copia tus imágenes a estos directorios
2. Mantén los mismos nombres de archivo
3. O actualiza las rutas en los componentes
